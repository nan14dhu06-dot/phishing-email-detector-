
# Phishing Email Detection Model

## 📌 Project Description
This project builds a machine learning model to detect phishing emails using Scikit-learn.

## 🚀 Features
- Train on phishing and safe emails
- Extract text features
- Classify emails as phishing or safe
- Display accuracy
- Show confusion matrix

## 🛠️ Technologies Used
- Python
- Scikit-learn
- Pandas
- Numpy
- Matplotlib
- Seaborn
